# Security Response Process
#
# 1. Report vulnerabilities via GitHub Security Advisories
# 2. We acknowledge within 48 hours
# 3. We provide status updates every 5 business days
# 4. We aim to fix within 90 days
# 5. Coordinated disclosure with reporter
#
